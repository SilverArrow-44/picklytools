How to use

Your city loads automatically alongside New York, London, Dubai, Tokyo, and Sydney as examples. Add or remove any city using the search box. The timeline shows what local time each column is for each city — so you can see at a glance that 9am in London is 4pm in Dubai and 11pm in Tokyo.

Drag across the hour blocks to select a meeting window. The detail panel shows the exact start and end time in every city. Use ½hr slots for 30-minute precision.

Frequently asked questions

Why do the rows show different times for the same column?
That is the whole point. Each column is the same moment in time, shown in each city's local clock. So column 9a (9am your time) will show a different time in every row depending on the timezone.
How do I select a multi-hour meeting slot?
Just drag. Press on your start block and drag right to your end block. The whole range highlights as you drag, and the detail panel shows start and end times for all cities.
How do I get half-hour precision like 9:30am?
Click the ½hr slots button. Each hour splits into two 30-minute blocks so you can pick 9:00 or 9:30 separately.
Can I change my city if it was detected wrong?
Yes. Hover over any card and click the X to remove it, then search for your actual city using the Add a city button.
Does it handle Daylight Saving Time?
Yes. The browser's Intl API uses your OS timezone database, which handles DST correctly for all regions.
